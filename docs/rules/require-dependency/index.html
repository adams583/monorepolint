:require-dependency

Require all packages to declare dependencies with specified versions.

Often useful when using a third party tool like webpack or babel that expects a local entry.

Options#

  • dependencies (Optional)
    • Map of dependency name to version
  • devDependencies (Optional)
    • Map of dependency name to version
  • peerDependencies (Optional)
    • Map of dependency name to version
  • optionalDependencies (Optional)
    • Map of dependency name to version

Example#

module.exports = {
rules: {
":require-dependency": {
options: {
devDependencies: {
typescript: "^3.8.3",
},
},
},
},
};
